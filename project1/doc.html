Project Goal & Audience

The goal of this first project is to build a one-page landing site centered around the indie psychological horror game Fran Bow. For Phase 1, I extracted the structural layout from the provided RIT Athletics magazine page and replaced the content with "lorem ipsum" placeholder text and my own game assets to prepare for Phase 2. I applied a darker, atmospheric color palette suitable for my game's theme. The audience would be fans of narrative-driven indie games and psychological horror. I chose this specific layout because the large horizontal spaces are perfect for showcasing the beautiful, hand-drawn background art of the game.

Notes & Challenges

  • Grid & Columns: I used a CSS Grid setup grid-template-columns: 3fr 1frto split the text and sidebar. To achieve the magazine-style text layout within the left section, I utilized the CSS column-count: 3 property.
  • Image Formatting: Getting the images to display 100% of their content without being cropped required swapping from background images to native img tags and using the object-fit: contain CSS property.
  • Layering: Getting the text to overlap the top image properly required wrapping the image in a relative container and using absolute positioning for the text, satisfying the layering requirement.

Responsive Plan

When the screen width decreases below 900px, the grid collapses from a two-column setup to a single column (1fr), moving the right-side boxes directly underneath the main text block. The 3 text columns also reduce to 2 columns, and eventually to a single column on mobile screens (max-width 600px).
